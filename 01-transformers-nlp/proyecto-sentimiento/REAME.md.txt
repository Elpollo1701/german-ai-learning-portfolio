# 📊 Clasificador de Sentimiento con Transformers

Este proyecto forma parte de mi ruta de aprendizaje en inteligencia artificial. Aquí implemento un modelo de clasificación de sentimientos basado en Transformers, utilizando la biblioteca `transformers` de Hugging Face y conjuntos de datos públicos.

---

## 🚀 Objetivo

Desarrollar un modelo capaz de identificar el sentimiento (positivo, negativo, neutro) de textos breves, usando transfer learning con modelos preentrenados de Hugging Face.

---

## 🧰 Tecnologías y Herramientas

- Python 3.10+
- Transformers (Hugging Face)
- Datasets (IMDB o Yelp Reviews)
- Scikit-learn
- Pandas & NumPy
- Matplotlib / Seaborn
